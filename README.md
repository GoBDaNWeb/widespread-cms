# Widespread CMS

Фронтенд для админ-панели онлайн-магазина одежды.

## Tech Stack

- React
- Typescript
- Tanstack-router
- Tanstack-query
- Zustand

## Запуск

- Установка зависимостей

```sh
  npm i
```

- Запуск локально

```sh
  npm run dev
```

## Линтинг

- Проверить код

```sh
  npm run lint
```
