# Widespread CMS

Фронтенд для админ-панели онлайн-магазина одежды.
В основе — FSD-архитектура, которую я внедряю и на основной работе.

> ⚠️ Проект в разработке — реализована архитектура и логика авторизации пользователя, функциональность дополняется.

## Что реализовано

- FSD-архитектура (Feature-Sliced Design)
- Авторизация пользователя по JWT
- Базовая маршрутизация через Tanstack Router
- Управление серверным состоянием через Tanstack Query
- Небольшой бэкенд на FastAPI (находится в репозитории: [ссылка](https://github.com/GoBDaNWeb/widespread-backend))

## Tech Stack

**Frontend**

- React + TypeScript
- Tanstack Router — файловая маршрутизация
- Tanstack Query — работа с серверным состоянием
- Zustand — клиентское состояние

**Backend**

- FastAPI

## Запуск

Установка зависимостей:

```sh
npm i
```

Запуск локально:

```sh
npm run dev
```

## Линтинг
